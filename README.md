## Pre-requisites

Python3 modules:
gensim
nltk 
numpy
pandas
scikit-learn
spacy (only needed if requesting its tokeniser with `--tokeniser`)

NLTK asks to further run inside python:
```
import nltk
nltk.download('punkt')
```

Spacy instructions say to run
```
python -m spacy download en_core_web_sm
```
